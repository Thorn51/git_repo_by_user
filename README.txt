This is an assignment completed for the Thinkful Full Stack Flex program in the Asynchronous Web Apps module. It's purpose is to utilize the GitHub API to search for and display repositories by a users handle. 

Your team is working on an app that will help recruiters review all of a candidates information. You've been assigned to work on one feature for the app - to display a list of repos belonging to a particular GitHub handle.
    o	Review GitHub's API documentation for the list user repositories endpoint to understand how this endpoint works.
    o	Create an app and push it to GitHub.
    o	When you're done, submit the link to your GitHub repo at the bottom of the page.
